## Pre-Configuration Steps

Before shipping:

1. **Flash Raspberry Pi OS to SD card**
   - Use Raspberry Pi Imager
   - Enable SSH
   - Set username/password
   - **DO NOT configure WiFi** (using Ethernet)

2. **Boot Pi and SSH in**
   - Connect Ethernet adapter
   - Plug in power
   - Find IP in router
   - SSH in

3. **Run pre-configuration script**
   ```bash
   cd ~/prostat-bridge
   ./pre-configure-bridge.sh --static-ip 192.168.0.200
   ```

4. **Test everything**
   - Verify service is running
   - Test health endpoint
   - Check logs

5. **Create sticker**
   - Print from STICKER-INFO.txt
   - Attach to case

6. **Package**
   - Include all items
   - Add instruction sheet
